# Changelog

All notable changes to the "trad-stock-monitor" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-10

### Added
- Initial release of TRAD Stock Monitor extension
- Real-time A-share stock price monitoring
- Profit/loss calculation based on purchase price and quantity
- TreeView interface in VSCode sidebar
- Configuration management for multiple stocks
- Start/stop monitoring controls
- Manual refresh functionality
- Status bar integration
- Webview configuration interface
- Support for both Sina Finance API and akshare data sources
- Auto-start monitoring option
- Price alert threshold configuration
- Stock import/export functionality
- Configuration backup and restore
